# Renaissance-frontend

**Renaissance**, is ultimately a next-generation blockchain-powered sports and entertainment engagement platform.

APP DESCRIPTION:
Renaissance is a Web3 football fan engagement platform built on stellar with:
- Real-time match insights
- Footballer lifestyle content feed
- Fan challenges and interactive game modes
- NFT-based rewards and collectibles
- Decentralized predictions (powered by smart contracts)
- Player rankings and fan leaderboards
- Social interactions between fans

The app will display:
- A home dashboard with match insights
- A feed of premium lifestyle content
- A challenges screen with interactive cards
- A profile screen with stats
- A modal system for challenge details

TECH SETUP:
Create a full Expo React Native app using:
- Expo SDK 51+
- TypeScript
- NativeWind (Tailwind for RN)
- ShadCN UI for React Native (https://ui.shadcn.com/react-native)
- React Navigation (stack + tabs)
- Lucide Icons for React Native
- AsyncStorage for local persistence
